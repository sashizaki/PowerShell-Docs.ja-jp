# <a name="whats-new-in-powershell-docs-for-2020-q1"></a>2020 年第 1 四半期の PowerShell ドキュメントの最新情報

この記事では、先月ドキュメントに行われた主な変更点をいくつか紹介し、コミュニティからのコントリビューションに感謝の意を表します。
